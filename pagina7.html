FlexBox

Até 2008 o mecanismo de posicionamento existente no CSS eram o fluxo do HTML, flot e positions, mas em 2007 se popularizou dispositivos móveis. Para possibilitar um maior responsitividade o CSS 3 trouxe alguns recursos como Screen Media, que permite definir conjunto de regras diferentes para com base no tamanho da tela do usuário e também a propriedade display: flex, que especifica o mecanismo FlexBox de posicionamento dos elementos 

        <style>
        .container{display:flex}
        </style>
