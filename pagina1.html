Lista com comandos básicos do git

  • git init - Cria um repositório em uma pasta vazia
  • git add - Adiciona modificações do wk no stage
  • git status - Mostra o estado atual do repositório, modificações arquivos versionados
  • git commit - Cria um commit no repositório LOCAL
  • git push - Envia as modificações do repositório LOCAL para o remoto
  • git log - Exibe no terminal os commits do repositório na branch atual
  • git log --oneline --all - Exibe todos os commits de forma resumida
  • git branch - Lista as branches do repositório
  • git checkout - Muda de branch
  • git checkout -b <nome-da-branch> - Cria e muda para uma nova branch
  • git clone url(repositório) - Clona o repositório
  • git diff - Exibe a diferença entre versões do repositório
