O CSS documento de estilização em cascata criando em 1993, com sua primeira versão amplamente utilizada em 1997. A liguagem CSS permite definir definir regras que definem características para os elmentos HTML como cor, borda e comportamentos, além de animações e transições

Ao longo dos anos o css evoluiu e hoje temos 1 2 e 3, cada uma das versões adicionou novos recursos que eram demandados no período

  1. CSS 1 1996 - propriedades básicas de estilização e formatação, além de controle do posicionamento dos elementos html com display, propriedades float e position
  2. CSS 2 1998 - adição de propriedades de animação e transição e melhoria do position
  3. CSS 3 2008 - incremento gradual de novas funcionalidades do CSS com animação e ferramentas de posicionamento como display:flex(2008) e grid (2013)

Uma regra pode ser definida com a seguinte sintaxe

            Seletor{
                Propriedade1:Valor;
                Propriedade2:Valor;

                Propriedade3:Valo;

            }

Podemos incluir código CSS de 3 formas diferentes

CSS externo
Quando definimos as regras CSS a serem utilizadas por meio de um arquivo externo .css, viculando ele ao html por meio da tag < link > no head
CSS incorporado "no documento"
Quando definimos as regras CSS por meio da tag <style >
CSS inline
Quando definidos código css de estilização no próprio elemento HTML como: Texto de cor vermelha com CSS inline
